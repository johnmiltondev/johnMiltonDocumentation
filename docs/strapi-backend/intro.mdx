# Introducción

Este es el sistema principal de la información del backend y la base de datos.

La principal función de esta herramienta es poder modificar en tiempo real la
información que se encuentra sobre la web. Entre las cosas que se pueden modificar
estan:

- Imagenes
- Parrafos
- Titulos
- Videos
- Creación/eliminación/edición de productos
- etc.

Es importante que se lea la documentación y se sigan las recomendaciónes indicadas
en esta página de documentación, ya que de no ser así, al momento de modificar secciones
de la web y contar con un error, puede que la página presente problemas.

Se intento ser lo mas detallado posible, si aún asi se tienen dudas, por favor
haganle saber al developer de confianza para que los apoye.
