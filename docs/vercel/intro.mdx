# Introducción

Vercel es la página de hosteo para el frontend. Aqui es donde todo lo relacionado
a la vista grafica de la web se hostea.

En Strapi Cloud se podran realizar deploys de la aplicacion frontend, revisar
el estatus de las request totales, el espacio en memoria de los archivos de media,
manejar los pagos del frontend, crear nuevos ambiente en caso de necesitar de esto, etc.

:::danger

Es importante que a Strapi Cloud solo tenga acceso el dueño del proyecto o algún desarrollador de
confianza, ya que darle acceso a alguien mas pondra en riesgo la base de datos y el
funcionamiento de la web.

:::
