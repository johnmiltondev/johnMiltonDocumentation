# Perfil de la cuenta

Para acceder al perfil de la cuenta solo es necesario ingresar a traves de la imagen circular
superior de la derecha.

<div style={{textAlign: 'center'}}>
  <img src="/img/docs/strapi-cloud/strapi-cloud-profile.png" alt="strapi cloud profile page" />
</div>

## General

Esta pestaña muestra datos generales como el nombre, la cuenta con la que se inicia sesion y un boton para eliminar la cuenta.

:::danger
Importante no hacer click en eliminar la cuenta, esto hara que el proyecto deje de funcionar
y los datos se eliminaran.
:::

<div style={{textAlign: 'center'}}>
  <img src="/img/docs/strapi-cloud/strapi-cloud-profile-general.png" alt="strapi cloud profile general" />
</div>

## Cobros

Esta es la parte donde se ingresa el metodo de pago para los pagos recurrentes del servicio de backend. De igual manera cuenta con una sección para ingresar la dirección de facturación.

<div style={{textAlign: 'center'}}>
  <img src="/img/docs/strapi-cloud/strapi-cloud-profile-billing.png" alt="strapi cloud profile cobros" />
</div>

## Recibos

Por ultimo, la pestaña de invoices mostrara los recibos de pago recurrentes. Estos recibos se
pueden descargar de manera local si asi se desea.

<div style={{textAlign: 'center'}}>

</div>
