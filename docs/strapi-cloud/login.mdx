# Login

Strapi Cloud permite solamente ingresar con cuentas de Github, Gitlab y Google.
La cuenta relacionada al proyecto fue linkeada a traves de Github, debido a que
ahi se encuentra nuestro codigo fuente. Por esa razón sera necesario ingresar
a traves de esa opción.

<div style={{textAlign: 'center'}}>
  <img src="/img/docs/strapi-cloud/strapi-cloud-login.png" alt="strapi cloud login page" />
</div>

Si se ingresa a traves de un navegador en el que nose ha iniciado sesion antes o si ha pasado mucho tiempo desde la ultima vez que se ingreso entonces Github pedira que se verifique la cuenta a traves de
un codigo que llegara a la cuenta de Gmail de developers de John Milton.

# Pagina de inicio

La pagina de inicio aparece justo despues de iniciar sesion.
Esta pagina sirve para seleccionar el proyecto que tienes alojado en
Strapi Cloud.

Por el momento solo tenemos uno, por lo que solo tenemos que hacer
click sobre el nombre del proyecto para ingresar a la vista de detalle.
