# Introducción

Strapi Cloud es una herramienta de hosteo en la nube especializada para
alojar proyectos de Strapi.

En Strapi Cloud se podran realizar deploys de la aplicacion backend, revisar
el estatus de las request totales, el espacio en memoria de los archivos de media,
manejar los pagos del backend, crear nuevos ambiente en caso de necesitar de esto, etc.

:::danger

Es importante que a Strapi Cloud solo tenga acceso el dueño del proyecto o algún desarrollador de
confianza, ya que darle acceso a alguien mas pondra en riesgo la base de datos y el
funcionamiento de la web.

:::
